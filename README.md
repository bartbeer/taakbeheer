# taakbeheer

het doel van deze app is via de CLI taken te kunnen beheren. er is een database aan gekoppeld die de taken bijhoudt

zorg dat de folder taakbeheer noemt
om te starten maak folder taakbeheer en in de folder boven taakbeheer: in de cmd typ: "py -m taakbeheer.main" of python -m taakbeheer.main

export files worden gesaved in de folder boven taakbeheer.
